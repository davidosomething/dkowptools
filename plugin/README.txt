Add browser resolution to your admin bar.

== Description ==

Please report issues, fork, etc. from the GitHub repository:
https://github.com/davidosomething/dkowptools

Adds browser resolution to your admin bar.
There's an option to add it to a root node if you're using my dkowpconfig
setup, but no admin panel/gui for it yet.

== Installation ==

1. Install and activate this plugin through the 'Plugins' menu in WordPress

OR

1. Upload the `dkowptools` folder to your plugins directory
1. Activate the plugin through the 'Plugins' menu in WordPress
